\subsection{Objetivos}

\subsubsection{Objetivo Geral}
Desenvolver uma plataforma \textit{web} de transparência política que centralize, organize e simplifique o acesso aos dados públicos da Câmara dos Deputados, fomentando a fiscalização cidadã e o debate qualificado sobre a atuação parlamentar.

\subsubsection{Objetivos Específicos}
\begin{itemize}
    \item Implementar uma arquitetura de \textit{microservices} em Golang para ingestão, processamento e disponibilização escalável de dados legislativos;
    \item Desenvolver rotinas de ETL (\textit{Extract, Transform, Load}) para integrar dados da API RESTful v2 da Câmara e arquivos históricos em massa;
    \item Construir uma interface \textit{front-end} responsiva e acessível (WCAG 2.1) utilizando Next.js, facilitando a visualização de despesas e votações por cidadãos leigos.
\end{itemize}
